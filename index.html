Sobre Listenlist

Como amantes de la música, siempre buscamos nuevos álbumes, canciones, y artistas desde toda fuente posible: conocidos, redes sociales, blogs de música, criticos de música, y muchas más. Miles y miles de canciones son lanzadas cada día, haciendo el mercado musical uno de muy gran alcance, siempre hay algo para todo tipo de audiencia. Esta gran facilidad y accesibilidad a la música puede ser tan bueno como puede ser abrumador al consumidor. No siempre estamos dispuestos a escuchar de inmediato, ni tenemos la capacidad o memoria para recordar toda recomendación, pero queremos guardar esos descubrimientos para el momento que sí queramos.


De ahí nace la idea de Listenlist. El propósito de esta aplicación es dar una plataforma a los usuarios para coleccionar aquellos descubrimientos que dejamos pendientes. Al momento de querer sentarse y escuchar algo nuevo, Listenlist tiene al alcance esa colección preparada para escuchar. Esta aplicación no es un proyecto oficial de Spotify, pero hace uso del API público para generar todos los datos a usar.

Buscador


Encuentra tus artistas, álbumes y canciones favoritos utilizando la barra de búsqueda. Esta utiliza la base de datos de Spotify para asegurar una amplia cantidad de resultados.

Recomendaciones


Descubre recomendaciones de canciones similares a tus búsquedas para expandir tu lista y colecciones. Basado en encontrar canciones similares mediante diferentes propiedades de sonido, el algoritmo de Spotify analiza y encuentra canciones para agregar a tu lista.

Colecciones


Listenlist te permite crear colecciones para organizar tu lista de una manera que tenga sentido para ti, ya sea por género musical, artistas o incluso por estado de ánimo. También puedes ordenar fácilmente tu colección mediante diferentes propiedades como popularidad, duración o fecha de lanzamiento con un solo clic.

Accesibilidad


Dale un toque personal a la apariencia de la aplicación al cambiar el tema entre claro y oscuro, o también puedes modificar el tamaño de la fuente a tu gusto.

Preguntas Frecuentes

¿Se puede reproducir música?

No. Listenlist tiene el propósito principal de organizar una colección para escuchar después. Si Spotify es tu reproductor de elección, puedes encontrar links que te llevan directo al álbum, canción o artista desde tu lista.

¿Se necesita una cuenta de Spotify?

No. El uso de Spotify es simplemente como una referencia y base de datos, no es necesario para buscar y descubrir música, crear una lista, o utilizar la aplicación.

¿La aplicación es gratuita?

Sí, totalmente. Cualquier usuario puede utilizar el buscador y descubrir música, y para comenzar una lista el único requerimiento es registrar una cuenta.
